# OpenFoodFactsLite - Учебный проект на основе данных https://world.openfoodfacts.org/
(Архитектура - mvp + bulder + router)\
UIKit\
AVFoundation\
URLSession\
NSCache

Приложение позволяет проводить поиск продукта по названию и по штрих-коду.
